# 📚 SparkShare

A modern Django-based platform designed for college students to upload, share, and access academic resources such as notes, PDFs, question papers, assignments, and study materials.

SparkShare helps students collaborate and learn efficiently by creating a centralized repository of educational content.

---

## 🚀 Features

### 👤 User Authentication

* User Registration
* Secure Login & Logout
* OTP Verification
* Password Reset via Email

### 📄 Resource Sharing

* Upload PDF Notes
* Upload Study Materials
* Categorized Resources
* Download Shared Files

### 🔍 Easy Access

* Browse Resources
* Search Educational Materials
* View Resource Details

### 👨‍🎓 Student Profiles

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/shivakumar-karnati/sparkShare.git
cd sparkShare
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run Server

```bash
python manage.py runserver
```
